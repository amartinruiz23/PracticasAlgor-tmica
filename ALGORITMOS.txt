SELECCIÓN

	- Buscar el mínimo elemento
	- Intercambiarlo con el primero
	- Y así con todos

BURBUJA

	- Coges la primera pareja del vector
	- El mínimo lo pones primero
	- Y así con cada pareja
	- Repetimos el proceso tantas veces como sea necesario

INSERCIÓN

	- Se divide el vector en ordenados y no ordenados. Al principio el vector ordenados esta formado por ningún elemento.
	- Se mete el primer elemento a los ordenados
	- Para cada elemento del vector desordenado se inserta en el vector ordenado donde le corresponda.

MERGESORT

	- DIVIDE Y VENCERÁS
	- SE DIVIDE EL VECTOR EN SUBLISTAS CADA VEZ MÁS PEQUEÑAS HASTA DE TAMAÑO 2
	- LUEGO SE RECONSTRUYE HACIA ARRIBA (ÁRBOL)
